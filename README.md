## Overview

This project demonstrates the development of an interactive **business intelligence dashboard** using **R Shiny**.

The dashboard enables users to analyse retail sales performance across different **regions** and **years** through dynamic filtering, KPI reporting, and interactive visualisations. It was designed to transform transactional sales data into an intuitive decision-support tool for exploring business performance and profitability.

## Dashboard Features

* Executive KPI dashboard
* Sales performance analysis
* Profitability analysis
* Customer segment analysis
* Monthly trend analysis
* Regional and yearly filtering
* Interactive visualisations
* Searchable data tables

# Superstore BIダッシュボード（R Shiny）

## 概要

本プロジェクトでは、**R Shiny**を用いて、小売業の売上データを分析するインタラクティブなBIダッシュボードを開発しました。

地域や年度を自由に切り替えながら、売上、利益、顧客セグメントなどの主要指標をリアルタイムで分析できるよう設計しています。経営指標を直感的に把握し、データに基づく意思決定を支援することを目的としています。

### **デモアプリ**
https://thequeenofdata.shinyapps.io/Superstore/

## 主な機能

* KPIダッシュボード
* 売上分析
* 利益分析
* 顧客セグメント分析
* 月別トレンド分析
* 地域・年度フィルター
* インタラクティブな可視化
* 検索可能なデータテーブル

## ダッシュボード概要

売上、利益、顧客セグメント、地域別の業績をインタラクティブに分析できるBIダッシュボードを構築しました。利用者は地域や年度を自由に切り替えながら分析を行うことができ、データに基づく意思決定を支援します。
